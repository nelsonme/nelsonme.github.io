Interactive Leaflet-based Maps

Inspired by and modified from examples on leafletjs.com

Display User Location

A fullscreen map that shows the user's current location. Suitable for mobile devices or the desktop. This version does not update automatically. If the user has moved, they have to refresh the page to see their new location.

Follow User Location

As above, but automatically update marker to show user's current location.
