rand.js - random number generation (source)
  • uses LCG algorithm if user specifies a seed, otherwise uses Math.random() because it is faster
  • rand() - uniform random number in [0,1)
  • randa(array) - returns a random element from the array
  • randf([a],[b]) - uniform random float
    • randf() - random float in [0,1); same as calling rand()
    • randf(a) - random float in [0,a)
    • randf(a,b) - random float in [a,b)
  • randi([a],[b]) - returns random integer
    • randi() returns 0 or 1
    • randi(a) returns random integer in [0,a)
    • randi(a,b) returns random integer in [a,b)
  • randn([std=1]) - normally distributed random values with zero mean and specified std (default=1)
  • seed(val) - sets the seed and starts using LCG algorithm
  • seed() - returns the current seed
  • unseed() - switches back to unseeded mode, starts using Math.random()
  • isSeeded() - returns seeded status (true/false)
util.js - utility functions for arrays (source)
  • amin(array) - min value of array
  • amax(array) - max value of array
  • amean(array) - mean value of array
  • astd(array) - standard deviation of array
  • aequals(a, b) - returns true if arrays a and b are equal
    • only works if all elements are primitives (numbers, strings, booleans)
  • arange([start,] stop, [step=1]) - returns an array of evenly spaced values
    • the stop point is NOT included
    • arange(0, 3, 0.5) returns [0, 0.5, 1.0, 1.5, 2.0, 2.5]
  • hist(array, bins) computes histogram counts in specified bins
    • bins is an array of lower bin edges, equally spaced
    • returns {counts, underflow, overflow}, counts is same size as bins, underflowand overfloware scalars
  • linspace(start, stop, [N=20], [endpt = true]) - returns an array of N evenly spaced numbers
    • the stop point IS included by default (controlled by endpt flag)
    • (useful for generating exactly N bins for hist)
