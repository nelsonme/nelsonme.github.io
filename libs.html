util.js - utility functions for arrays
  • amin(array) - min value of array
  • amax(array) - max value of array
  • amean(array) - mean value of array
  • astd(array) - standard deviation of array
  • aequals(a, b) - true if arrays a and b are equal (simple elements, like numbers)
  • arange([start,] stop, [step=1]) creates an array of evenly spaced values
    • the stop point is NOT included
    • arange(5) returns [0, 1, 2, 3, 4]
    • arange(6, 10) returns [6, 7, 8, 9]
    • arange(0, 3, 0.5) returns [0, 0.5, 1.0, 1.5, 2.0, 2.5]
  • hist(data, bins) computes histogram counts in specified bins
    • bins is an array of lower bin edges, equally spaced
    • returns {counts, underflow, overflow} - counts is same size as bins, underflow and overflow are scalars
  • linspace(start, stop, N=20, endpt = true) creates an array of N evenly spaced numbers
    • the stop point IS included by default (controlled by endpt flag)
    • linspace(0, 5, 5) returns [0, 1.25, 2.5, 3.75, 5]
    • linspace(0, 5, 5, false) returns [0, 1, 2, 3, 4]
    • linspace(0, 5, 6) returns [0, 1, 2, 3, 4, 5]
